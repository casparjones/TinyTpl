'TinyTpl

Über die Diskusion ob für PHP eine Template Sprache genutzt wird oder PHP 
als eigene Template Sprache verwendet wird, entstand diese "Mini-Template"
was eigentlich kein eigene Template-Engine ist, sondern einfach nur
den "include" des PHP Files in eine Klasse Kapselt und die Variablen
über die Methode ->assign(key, value) (ähnlich wie bei Smarty) an das
Template weiterreicht.

Ideen für PHP als Templatesprace:
